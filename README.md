#  Carbon Footprint Calculator

A modern and responsive web application that helps users calculate their carbon footprint based on lifestyle, travel habits, and household choices. The platform educates users on climate impact, forest conservation, and how sustainable living can protect our planet.

Built using **React + Vite + Tailwind CSS**, the project is optimized for **speed**, **responsiveness**, and **clean UI** across both mobile and desktop devices.

---

##  Features

-  Calculate **individual** carbon footprint  
-  Calculate **household** carbon footprint  
-  Interactive and easy-to-use UI  
-  Educational sections on:
  - How carbon offsetting works  
  - Forest conservation  
  - How local communities benefit  
-  Fully responsive for mobile, tablet, and desktop  
-  Fast performance with **Vite + React**

---

##  Tech Stack

- **React.js**  
- **Vite**  
- **Tailwind CSS**  
- **React Router DOM**  
- **JavaScript (ES6+)**
